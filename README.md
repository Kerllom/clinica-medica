# 🏥 Sistema de Clínica Médica

Sistema para agendamento de consultas médicas, onde o paciente pode escolher
especialidade, médico, dia e horário. Projeto desenvolvido para a disciplina
de [NOME DA DISCIPLINA].

## ✨ Funcionalidades

- Cadastro de pacientes
- Login e autenticação de usuários
- Agendamento de consultas (especialidade → médico → dia → horário)
- Verificação de disponibilidade de horários
- Cancelamento e remarcação de consultas
- Listagem das consultas agendadas
- Gerenciamento de médicos e especialidades (administrador)
